tehtävien palautukset.


Alla olevat testit epäonnistuvat, koska tehtävien edetessä koodi on niiden osalta muuttunut ja autentikointi on tullut mukaan:

*POST /api/blogs adds one blog with valid content*
*POST /api/blogs with no likes, return 0 likes*
*POST /api/blogs with no title or URL, returns 400*
*DELETE /api/blogs/:id deletes blog if id is valid*
